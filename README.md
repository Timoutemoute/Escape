# 🚨 ESCAPE GAME SNT - Protocole ARGOS

Bienvenue dans l'escape game SNT !  
Vous devez désactiver un timer en trouvant le code final.

## 🎮 Comment jouer

1. Ouvrez `index.html` dans votre navigateur
2. Complétez les 4 épreuves dans l'ordre

## 🔍 Où trouver le code final ?

Le code final n'est pas caché dans le jeu.

**Indice :**  
Regardez attentivement le fichier `script.js`...  
Il contient un tableau `SOLUTIONS` avec les réponses des épreuves.

Mais attention, la solution de l'épreuve 4 n'est pas la bonne clé pour arrêter le timer !
