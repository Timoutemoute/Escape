🔌 COMMUTATEUR

Étiquette technique :

"Adresse réseau = Adresse IP AND Masque"

"42 en binaire = 00101010"

Rappel : Masque /27 = 255.255.255.224

Pour trouver l'adresse réseau :

IP 192.168.10.42 AND 255.255.255.224 = 192.168.10.32

Dernier octet = 32 → binaire = 00100000
